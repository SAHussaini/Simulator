PROJECT TITLE: Supermarket Simulator
AUTHOR: Syed A. Hussaini

PURPOSE OF PROJECT: 
Simulate a supermarket customer entry-exit flow under certain entry regulations

USER INSTRUCTIONS:
This folder contains three folders:
1. Class Code: contains code of the three classes used in the project (i.e. Controller, Supermarket and Customer)
2. Test Code: contains the test code for each of the three classes used in the project
3. Contains the BlueJ package as well as copies of all of the above code (so that BlueJ can find the code relating to the project)
